# Package index

## Main Functions

Primary functions for working with COMPASS output

- [`plot_compass()`](https://satvilab.github.io/UtilsCompassSV/reference/plot_compass.md)
  : Plot COMPASS output
- [`convert_cyt_combn_format()`](https://satvilab.github.io/UtilsCompassSV/reference/convert_cyt_combn_format.md)
  : Convert cytokine combination format to or from COMPASS format
- [`response_prob()`](https://satvilab.github.io/UtilsCompassSV/reference/response_prob.md)
  : Calculate COMPASS-derived overall responder probability

## Helper Functions

Supporting functions for plotting and layout

- [`.save_layout()`](https://satvilab.github.io/UtilsCompassSV/reference/dot-save_layout.md)
  : Create a layout based on post probs, cyt heat map and scores
